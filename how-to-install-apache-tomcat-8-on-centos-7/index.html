介绍

Apache Tomcat是一个Web服务器和servlet容器，用于为Java应用程序提供服务。Tomcat是Apache Software Foundation发布的Java Servlet和JavaServer Pages技术的开源实现。本教程介绍CentOS 7服务器上Tomcat 8的最新版本的基本安装和一些配置。
